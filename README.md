# 📚 Once Upon a Book  

**Once Upon a Book** é uma aplicação de **livraria online**, criada como projeto da disciplina **Linguagem de Script (LS)**.  
O sistema foi iniciado em **JavaScript, HTML e Tailwind** e está sendo migrado para **React**. 

## 🛠 Tecnologias  
- JavaScript  
- React (em migração)  
- TailwindCSS  
- HTML5
